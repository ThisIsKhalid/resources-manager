Phase 4 — Pro ML Engineer
Duration: Ongoing · Specialization and scale
Spec
Pick a specialization
  • NLP / LLMs — most in-demand right now
  • Computer Vision — strong in robotics, medical, manufacturing
  • Recommender systems — e-commerce, streaming, social
  • Reinforcement Learning — robotics, game AI, RLHF for LLMs
  • Time series / forecasting — finance, logistics, ops
SysD
ML system design
  • Design a recommendation system at scale
  • Online vs offline feature stores
  • Two-tower models, candidate retrieval
  • ML design interview patterns
Port
Build your portfolio
  • 3–5 strong GitHub ML projects with demos
  • Kaggle — aim for Expert or Master ranking
  • Write papers or posts on Medium / Arxiv
  • Contribute to PyTorch or Hugging Face
